3D Optical System Simulator

An interactive 3D ray-optics simulation. Place optical objects, set their geometry and material properties, and trace light paths through the system in real time.

How to use
01 Create an object
Use the "Create Object" control in the simulation to add a new optical element (lens, mirror, prism, etc.) to the scene.
02 Select an object
Click the object selector to choose which placed object you want to configure. All property controls apply to the currently selected object.
03 Set position & rotation
Enter X, Y, Z coordinates to position the object in 3D space. Use the rotation fields (X / Y / Z) to orient it — values are in degrees.
04 Set dimensions
Adjust Length, Height, and Width to scale the optical element. Units are in the simulation's internal coordinate system.
05 Set refractive index
Enter the refractive index (n) of the material. Common values: air ≈ 1.0, glass ≈ 1.5, diamond ≈ 2.42, you can create an opaque object by setting this parameter to 0. Overlapping objects take on the average refractive index, you can use this to create hollow objects.
Parameter reference
Param Description
x, y, z World-space position of the object's centre
rx, ry, rz Rotation around each axis (degrees)
length Extent along the object's local X axis
height Extent along the object's local Y axis
width Extent along the object's local Z axis
n Refractive index of the material
