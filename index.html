GlowScript · Ray Optics

3D Optical System Simulator

An interactive 3D ray-optics simulation. Place optical objects, set their geometry and material properties, and trace light paths through the system in real time.

↗ Open in GlowScript
3DOptSimLTJ - glowscript.org
How to use
01 Create an object
Use the "Create Object" control in the simulation to add a new optical element (lens, mirror, prism, etc.) to the scene.
02 Select an object
Click the object selector to choose which placed object you want to configure. All property controls apply to the currently selected object.
03 Set position & rotation
Enter X, Y, Z coordinates to position the object in 3D space. Use the rotation sliders (X / Y / Z) to orient it — values are in degrees.
04 Set dimensions
Adjust Length, Height, and Width to scale the optical element. Units are in the simulation's internal coordinate system.
05 Set refractive index
Enter the refractive index (n) of the material. Common values: air ≈ 1.0, glass ≈ 1.5, diamond ≈ 2.42. Set to 0 to create an opaque object. Overlapping objects take on the average refractive index — use this to create hollow objects.
06 Navigate the scene
Hold down right click to move around the scene. Use your scroll wheel to zoom in and out.
Parameter reference
ParamDescription
x, y, zWorld-space position of the object's centre
rx, ry, rzRotation around each axis (degrees)
lengthExtent along the object's local X axis
heightExtent along the object's local Y axis
widthExtent along the object's local Z axis
nRefractive index — 0 = mirror, <0 = absorber, >0 = refractive material
Tips & tricks
💡 Total internal reflection
Increase the refractive index and rotate the object until rays stop passing through — this is total internal reflection occurring at the critical angle.
💡 Mirrors
Set n = 0 to make an object behave as a perfect mirror. Rays hitting the surface will reflect rather than refract.
💡 Opaque absorbers
Set n to any negative value to make an object fully absorb all light that hits it. Useful for creating apertures, baffles, or beam stops.
💡 Hollow objects
Overlap two objects with refractive indices n₁ and n₂. The overlapping region takes on the average (n₁ + n₂) / 2, letting you simulate hollow or composite geometries.
Limitations
⚠ Rays inside objects
The simulation breaks down if a ray is created inside a refractive object and allowed to exit it. Always place ray sources in open space.
⚠ Interaction limit
Each beam is cut short after 100 interactions (reflections or refractions). Complex scenes with many overlapping objects may hit this limit before rays fully exit the system.
⚠ Performance
Performance decreases with every added object and interaction. Results will vary by device — a decent PC can handle upwards of 10,000 interactions, while a phone may struggle with far fewer. If the simulation slows down, try reducing the number of objects or rays.
